# <a name="compiler-error-cs0001"></a>Erro do Compilador CS0001

Erro interno do compilador

 Tente determinar se o compilador está falhando em razão de sua incapacidade de analisar sintaxes inesperadas. Se esse erro ocorrer repetidamente, entre em contato com a Microsoft.

# <a name="compiler-error-cs0214"></a>Erro do Compilador CS0214

Ponteiros e buffers de tamanho fixo só podem ser usados em um contexto sem segurança  
  
 Os ponteiros só podem ser usados com a palavra-chave [não segura](../language-reference/keywords/unsafe.md) . Para obter mais informações, consulte [Código não seguro e ponteiros](../programming-guide/unsafe-code-pointers/index.md).  
  
 O exemplo a seguir gera CS0214:  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```

# <a name="compiler-error-cs0448"></a>Erro do Compilador CS0448

O tipo de retorno para o operador + + ou--deve ser do tipo recipiente ou derivado do tipo recipiente  
  
 Quando você substitui os `++` `--` operadores ou, eles devem retornar o mesmo tipo que o tipo recipiente ou retornar um tipo que é derivado do tipo recipiente.  
  
## <a name="example"></a>Exemplo  

 O exemplo a seguir gera CS0448.  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>Exemplo  

 O exemplo a seguir gera CS0448.  
  
```csharp  
// CS0448_b.cs  
public struct S  
{  
   public static S? operator ++(S s) { return new S(); }   // CS0448  
   public static S? operator --(S s) { return new S(); }   // CS0448  
}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```

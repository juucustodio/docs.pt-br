# <a name="compiler-error-cs1551"></a>Erro do Compilador CS1551

Os indexadores devem ter pelo menos um parâmetro  
  
 Um [indexador](../programming-guide/indexers/index.md) que não usa argumentos foi declarado.  
  
 O exemplo a seguir gera CS1551:  
  
```csharp  
// CS1551.cs  
public class MyClass  
{  
   int intI;  
  
   int this[]   // CS1551  
   // try the following line instead  
   // int this[int i]  
   {  
      get  
      {  
         return intI;  
      }  
      set  
      {  
         intI = value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

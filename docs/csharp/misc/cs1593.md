# <a name="compiler-error-cs1593"></a>Erro do Compilador CS1593

' Del ' delegate não pega argumentos ' Number '  
  
 O número de argumentos passados para uma invocação [delegate](../language-reference/builtin-types/reference-types.md) não concorda com o número de parâmetros na declaração delegate.  
  
 O exemplo a seguir gera CS1593:  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```

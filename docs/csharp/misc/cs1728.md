# <a name="compiler-error-cs1728"></a>Erro do Compilador CS1728

Não é possível associar o delegado a ' member ' porque ele é membro de ' type '  
  
 Não é possível associar delegados a membros de `Nullable` tipos de valor.  
  
## <a name="example"></a>Exemplo  

 O exemplo a seguir gera CS1728:  
  
```csharp  
// CS1728.cs  
class Test  
{  
   delegate T GetT<T>();  
   delegate T GetT1<T>(T t);  
  
   delegate bool E(object o);  
   delegate int I();  
   delegate string S();  
  
   static void Main()  
   {  
      int? x = null;  
      int? y = 5;  
  
      GetT<int> d1 = x.GetValueOrDefault;   // CS1728  
      GetT<int> d2 = y.GetValueOrDefault;   // CS1728  
      GetT1<int> d3 = x.GetValueOrDefault;   // CS1728  
      GetT1<int> d4 = y.GetValueOrDefault;   // CS1728  
   }  
}  
```

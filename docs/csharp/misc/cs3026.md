# <a name="compiler-warning-level-1-cs3026"></a>Aviso do compilador (nível 1) CS3026

O campo ' Field ' em conformidade com CLS não pode ser volátil

Uma variável volátil não deve ser compatível com CLS.

## <a name="example"></a>Exemplo

O exemplo a seguir gera CS3026.

```csharp
// CS3026.cs
[assembly:System.CLSCompliant(true)]
public class Test
{
    public volatile int v0 =0;   // CS3026
    // To resolve remove the CLS-Compliant attribute.
    public static void Main() { }
}
```

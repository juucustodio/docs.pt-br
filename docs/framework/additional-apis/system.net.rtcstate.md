# <a name="rtcstate-class"></a>Classe RtcState

A `RtcState` classe representa dados de estado para uma solicitação de comunicação em tempo real (RTC).

## <a name="syntax"></a>Syntax
  
```csharp  
internal class RtcState
```

> [!WARNING]
> A `RtcState` classe é interna e não deve ser usada diretamente no seu código.
>
> A Microsoft não oferece suporte ao uso dessa classe em um aplicativo de produção em nenhuma circunstância.

## <a name="requirements"></a>Requisitos

**Namespace:** <xref:System.Net>

**Assembly:** Sistema (em System.dll)

**.NET Framework versões:** Disponível desde 4,5.

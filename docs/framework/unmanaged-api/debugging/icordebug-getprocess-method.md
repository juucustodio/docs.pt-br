# <a name="icordebuggetprocess-method"></a>Método ICorDebug::GetProcess

Obtém um ponteiro para a instância "ICorDebugProcess" para o processo especificado.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT GetProcess (  
    [in] DWORD               dwProcessId,  
    [out] ICorDebugProcess   **ppProcess  
);  
```  
  
## <a name="parameters"></a>Parâmetros  

 `dwProcessId`  
 no A ID do processo.  
  
 `ppProcess`  
 fora Um ponteiro para o endereço de uma `ICorDebugProcess` instância do processo especificado.

# <a name="failfastexception"></a>FailFastException

ID: 103  
  
 Gravidade: Erro  
  
 Categoria: rastreamento  
  
## <a name="description"></a>Descrição  

 Uma exceção foi lançada durante FailFast. O evento lista a exceção, o nome do processo e a ID do processo.

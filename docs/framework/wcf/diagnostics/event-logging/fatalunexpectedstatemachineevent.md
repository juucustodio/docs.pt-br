# <a name="fatalunexpectedstatemachineevent"></a>FatalUnexpectedStateMachineEvent

ID: 137  
  
 Gravidade: Erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  

 Esse evento indica que uma máquina de estado tentou processar um evento inesperado. O evento foi considerado fatal. O evento lista a ID da transação, a máquina de estado, o estado atual, o histórico, o nome do evento, os detalhes do evento, o nome do processo e a ID do processo.

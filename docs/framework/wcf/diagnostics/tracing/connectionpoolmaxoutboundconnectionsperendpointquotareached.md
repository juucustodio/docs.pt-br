System.ServiceModel.Channels.ConnectionPoolMaxOutboundConnectionsPerEndpointQuotaReached  
  
## <a name="description"></a>Descrição  

 A cota de MaxOutboundConnectionsPerEndpoint foi atingida, portanto, a conexão foi fechada e não está armazenada nesse pool de conexões.

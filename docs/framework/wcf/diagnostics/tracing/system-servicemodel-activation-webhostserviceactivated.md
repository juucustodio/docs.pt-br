# <a name="systemservicemodelactivationwebhostserviceactivated"></a>System.ServiceModel.Activation.WebHostServiceActivated

System.ServiceModel.Activation.WebHostServiceActivated  
  
## <a name="description"></a>Descrição  

 Ocorre quando um serviço hospedado no Serviços de Informações da Internet (IIS) ou no WAS (serviço de ativação de processos do Windows) é totalmente ativado e começa a escutar mensagens.

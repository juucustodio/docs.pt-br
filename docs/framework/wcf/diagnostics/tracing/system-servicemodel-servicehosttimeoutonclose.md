# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose

System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Descrição  

 A operação de fechamento de um ServiceHost atingiu o tempo limite. Isso indica que alguns canais de sessão não foram corretamente desligados pelo cliente e poderiam ter sido anulados. Modifique a hora padrão para ajustar o tempo limite de fechamento.

# <a name="behavior-class"></a>Behavior class

## <a name="syntax"></a>Syntax  
  
```csharp
class Behavior  
{  
  string Type;  
};  
```  
  
## <a name="methods"></a>Métodos  

 A classe de comportamento não define nenhum método.

 A classe de comportamento tem a seguinte propriedade:

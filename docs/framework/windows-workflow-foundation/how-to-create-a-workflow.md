# <a name="how-to-create-a-workflow"></a>Como: criar um fluxo de trabalho

Os fluxos de trabalho podem ser construídos a partir de atividades internas assim como as atividades personalizadas. Os tópicos nesta seção abordam a criação de um fluxo de trabalho que usa atividades internas, como a <xref:System.Activities.Statements.Flowchart> atividade, e as atividades personalizadas do tópico [como criar uma atividade](how-to-create-an-activity.md) anterior. O fluxo de trabalho modela um jogo de palpite de número. Somente um dos tópicos nesta seção é necessário para concluir o tutorial; você deve escolher o estilo que os interesses você e seguem essa etapa. No entanto, você pode concluir todos os tópicos se desejado.  
  
> [!NOTE]
> Cada tópico do tutorial de Introdução depende dos tópicos anteriores. Para concluir este tópico, você deve primeiro concluir [como: criar uma atividade](how-to-create-an-activity.md).

 Descreve como criar um fluxo de trabalho sequencial usando a atividade de <xref:System.Activities.Statements.Sequence> .

 Descreve como criar um fluxo de trabalho do fluxograma usando a atividade de <xref:System.Activities.Statements.Flowchart> .

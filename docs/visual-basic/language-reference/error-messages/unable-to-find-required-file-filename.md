# <a name="bc30655-unable-to-find-required-file-filename"></a>BC30655: não é possível localizar o arquivo necessário ' \<filename> '

 **ID do erro:** BC30655

## <a name="to-correct-this-error"></a>Para corrigir este erro

- Reinstale o Visual Studio.

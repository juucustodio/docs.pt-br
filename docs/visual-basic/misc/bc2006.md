# <a name="option-optionname-requires-argument"></a>A opção '\<optionname>' requer \<argument>

Um argumento necessário foi omitido.  
  
 **ID do erro:** BC2006  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Forneça o argumento ausente.

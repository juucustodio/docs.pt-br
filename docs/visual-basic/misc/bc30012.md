# <a name="if-block-must-end-with-a-matching-end-if"></a>Bloco '#If' deve encerrar com um '#End If' correspondente

`#If` é uma diretiva de compilação condicional. Um `#If` bloco não é encerrado por uma `#End If` diretiva.  
  
 **ID do erro:** BC30012  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Adicione uma `#End If` diretiva ao final do bloco de compilação condicional.

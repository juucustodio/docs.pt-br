# <a name="assembly-attribute-attributename-is-not-valid-error"></a>O atributo de assembly '\<attributename>' é inválido: \<error>

O Common Language Runtime não aceita o atributo em um assembly.

**ID do erro:** BC30129

## <a name="to-correct-this-error"></a>Para corrigir este erro

1. Remova o atributo da declaração ou substitua-o por um atributo que seja válido para o contexto.

2. Examine a mensagem de erro entre aspas e execute a ação apropriada.

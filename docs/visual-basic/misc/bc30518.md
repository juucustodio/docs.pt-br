# <a name="overload-resolution-failed-because-no-accessible-method-can-be-called-with-these-argumentserror"></a>Falha na resolução de sobrecarga porque nenhum '\<method>' acessível pode ser chamado com estes argumentos:\<error>

Você fez uma chamada para um método sobrecarregado, mas o compilador não pode localizar um método com uma lista de argumentos correspondentes.  
  
 **ID do erro:** BC30518  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique a lista de argumentos que você forneceu.

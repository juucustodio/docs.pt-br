# <a name="membername-is-not-declared-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>' \<membername> ' não está declarado ou o módulo que o contém não está carregado na sessão de depuração

O membro especificado não pôde ser encontrado.  
  
 **ID do erro:** BC30699  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Verifique se o membro está definido e escrito corretamente.  
  
2. Acessar um dos membros declarados no módulo. Em alguns casos, o ambiente de depuração não pode localizar membros porque os módulos em que eles estão declarados ainda não foram carregados.

# <a name="catch-statements-are-not-valid-in-the-immediate-window"></a>Instruções 'Catch' não são válidas na janela Imediata

`Catch` as instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30715  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remover `Catch` instruções do código de depuração.

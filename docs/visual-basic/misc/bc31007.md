# <a name="unable-to-open-module-file-filename-error"></a>Não é possível abrir o arquivo de módulo ' \<filename> ': \<error>

Não foi possível abrir um arquivo que contém um módulo para este projeto.  
  
 **ID do erro:** BC31007  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Verifique se o arquivo está no local especificado e não está bloqueado por outro aplicativo.

# <a name="duplicate-xml-attribute-attributename"></a>Atributo XML duplicado 'attributeName'

Um atributo especificado para um literal de elemento XML corresponde ao de um atributo que foi previamente definido para o literal de elemento XML. Os nomes de atributo em literais de elemento XML devem ser exclusivos.  
  
 **ID do erro:** BC31149  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Renomeie o atributo para um nome que seja exclusivo para o literal de elemento XML.

# <a name="bad-guid-format"></a>Formato de GUID inválido

Um GUID foi fornecido com um formato incorreto.  
  
 **ID do erro:** BC42035  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique o formato do GUID.
